Minitorch

Minitorch is a project that replicates the functionality of the popular machine learning library PyTorch, but in a smaller scale. It is a lightweight implementation that enables users to perform basic machine learning tasks such as building, training, and evaluating neural networks. It provides a simplified interface for running linear and convolutional neural networks.

Music Generation using LSTM

This music generation algorithm project uses LSTM neural network to learn patterns and structures from a MIDI dataset and generate new music. It enables the creation of original and unique music and has many potential applications in music production and entertainment industry.
